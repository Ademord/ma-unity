Package includes 3 unitypackage files. Each one designed for different Unity render pipeline:

1. Built-in - package for Built-in (Standard) render pipeline.
2. Universal - package for Universal scriptable render pipeline (URP).
3. High Definition - package for High Definition scriptable render pipeline (HDRP).

Based on a project’s render pipeline import one of the above packages.


Wireframe Shader minimum requirements are:
• Unity version 2019.4
• Universal RP 7.4.3
• High Definition RP 7.4.3
